# Day 18 Report: Natural Language Processing (NLP)

## Objective:
To understand and implement NLP techniques for processing unstructured text data, extracting user interests, and analyzing sentiment for the MeetMux matchmaking engine.

## Tasks Completed:
- Developed a text cleaning pipeline using tokenization, stopword removal, and text sanitization.
- Processed raw user bios into cleaner, machine-readable text.
- Implemented TF-IDF vectorization to convert bios into numerical feature matrices.
- Extracted important keywords representing user interests.
- Installed and used TextBlob for sentiment analysis.
- Performed sentiment analysis on 5 sample bios/reviews.
- Recorded polarity and subjectivity scores for each text sample.
- Documented the difference between stemming and lemmatization.

## Conclusion:
Successfully built a foundational NLP pipeline capable of cleaning text, extracting meaningful interests, vectorizing language into numerical form, and evaluating emotional tone for improved matchmaking and platform intelligence.
